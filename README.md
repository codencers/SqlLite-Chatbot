# AI SQL Copilot  
## Natural Language → SQL → Intelligent Data Interaction  

Talk to your database like you talk to ChatGPT.  
Powered by LangChain Agents + Groq LLM + Streamlit.

---

## Overview

AI SQL Copilot is an intelligent conversational interface that converts natural language into executable SQL queries.

Instead of writing complex SQL queries manually, users can simply ask questions in plain English and receive accurate database responses in real time.

This project demonstrates practical integration of:
- LLM Agents
- SQL Databases
- Real-time UI
- Production-oriented architecture
